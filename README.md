# 기술 블로그

Next.js와 Contentlayer를 활용한 개인 기술 블로그입니다.

## 기술 스택

- **Frontend**: Next.js
- **콘텐츠 관리**: Contentlayer
- **배포 및 인프라**:
  - AWS S3
  - AWS CloudFront
  - AWS Route53
- **CI/CD**: GitHub Actions를 통한 자동 배포

## 주요 기능

- Contentlayer를 활용한 마크다운 기반 블로그 포스팅
- AWS 인프라를 활용한 안정적인 서비스 제공
- GitHub 저장소에 푸시 시 자동으로 S3 업로드 및 배포
